# java-weather-api-app
Java Weather Application using REST API, HttpClient, and JSON parsing to fetch real-time weather data.

# Features

- Fetch real-time weather information
  
- REST API integration
  
- JSON data parsing
  
- Display temperature and weather conditions

- Error and exception handling
  
- User-friendly console output
  
#Technologies Used

- Java

- HttpClient
  
- REST API
  
- JSON Parsing
  
- IntelliJ IDEA
